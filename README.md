# ✨ Pixit — Your Developer Toolkit Companion

> Build faster. Code smarter. Enjoy the process.

---

## 🚀 What is Pixit?

**Pixit** is an all-in-one developer toolkit for Visual Studio Code that blends powerful productivity tools with a visually engaging, interactive experience.

It’s not just another extension.

It’s your **coding companion** — helping you generate code, clean your files, organize your project, and subtly guide you toward better development practices… all while feeling alive inside your editor.

---

## 🎯 Why Pixit?

Modern development is repetitive, messy, and sometimes overwhelming.

Pixit solves that by:

* ⚡ Automating repetitive tasks
* 🧠 Assisting with smarter code decisions
* 🧹 Keeping your codebase clean
* 🎨 Making your workspace feel dynamic and enjoyable

---

## 🧩 Core Features

### ⚡ Smart Code Generation

Generate production-ready code in seconds.

* Components (React, etc.)
* APIs & services
* Hooks and utilities
* Multi-file scaffolding

```bash
Pixit: Generate Component
```

Creates:

```
MyComponent/
 ├── MyComponent.tsx
 ├── MyComponent.css
 └── test.ts
```

---

### 🧹 Pixit Clean — Instant Code Cleanup

Keep your codebase tidy without thinking about it.

* Removes `console.log`
* Clears unnecessary whitespace
* Prepares code for production

```bash
Pixit: Clean File
```

Your tiny coding partner inside the editor.

Inspired by interactive tools like VS Code Pets, but designed for real productivity.

* ⚠️ Error — alerts you gently

## ⚡ Getting Started

1. Install Pixit (coming soon to marketplace)
2. Open Command Palette:

```
Ctrl + Shift + P
```

3. Try commands:

```
Pixit: Generate Component
Pixit: Clean File
Pixit: Open Companion
```
